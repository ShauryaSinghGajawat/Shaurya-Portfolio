About Me

Aspiring game environment artist aiming for DAE — Game Graphics Production Major.

What I Care About

  • Gameplay-friendly layout and clear navigation for the player.
  • Strong focal points and value structure to lead the eye.
  • Material definition and believable detail density.
  • Optimized assets for real-time engines like Unreal.

Skills

What I focus on when building worlds.

  • Composition and leading the eye with strong focal points.
  • Value planning in grayscale before adding texture detail.
  • Environment storytelling using props, decals and lighting.
  • Balancing detail density for readability in gameplay.
  • Modular level design for reusable environment kits.
  • Trim-sheet and tiling texture workflows.
  • Lightmap setup, basic optimization & LOD awareness.
  • Using Blueprints to support environment interactions.
  • Blockout → first lighting pass → detailing → polish.
  • Using reference boards and callouts before modeling.
  • Quick graybox iterations to test gameplay navigation.
  • Non-destructive material setups for easy iteration.
  • Taking feedback and applying it across multiple passes.
  • Documenting my decisions so designers understand intent.
  • Staying organized with naming, folders and versioning.
  • Always learning from ArtStation breakdowns & talks.
